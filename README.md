# The General Knowledge Quiz

## Project Description
This project is a simple command-line quiz application developed in Python.

The program:
- Asks 3 general knowledge questions.
- Accepts user input.
- Uses if-else statements for decision making.
- Maintains a score counter.
- Displays the final score at the end.

## Features
- Input sanitization using `.strip().lower()`
- Score tracking
- Immediate feedback for each answer
- Final performance summary

## Requirements
- Python 3.x

## How to Run

1. Save the code as `quiz.py`
2. Open a terminal in the project folder.
3. Run:

```bash
python quiz.py
```

## Example

```text
1. What is the capital of France? paris
Correct!

2. Which planet is known as the Red Planet? mars
Correct!

3. What is the largest ocean on Earth? pacific
Correct!

===== QUIZ COMPLETED =====
Final Score: 3/3
Excellent! You got all answers correct.
```

- User Input
